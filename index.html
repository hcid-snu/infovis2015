정보 비주얼라이제이션

Course Information

이 수업에서는 정보 비주얼라이제이션(information visualization)을 통하여 데이터가 가진 의미를 발견하고 사고의 확장을 도모한다. 현재 정보 비주얼라이제이션은 공학은 물론 통계학, 언론학, 인문학, 디자인 등의 다양한 분야에서 활발하게 진행되고 있는데, 이는 문제를 해결하고 의사 결정 과정에서의 비주얼라이제이션 방식이 데이터를 이해하고 복잡한 데이터에 의미를 부여하는데에 중요한 역할을 담당하고 있기 때문이다.

이 수업에서는 정보 비주얼라이제이션과 관련한 이론을 학습하고, d3.js와 R을 활용한 실습을 통해 정보 비주얼라이제이션을 제작한다.

강의 일정   Piazza

Assignments

1_ Codecademy

자바스크립트의 기초 학습

2_ SVG Drawing

SVG 태그를 이용한 간단한 그림 그리기 과제 (자바스크립트도 활용 가능)

3_ Bar Graph

Bar Graph를 이용한 데이터 분석 과제

4_ Pie+Line Graph

Pie Chart와 Line Graph를 이용한 데이터 분석 과제

5_ Network Graph

Network Graph를 이용한 데이터 연관 관계 분석 과제

6_ Tree/Heat Map

Tree Map과 Heat Map을 이용한 데이터 분석 과제

7_ Map Visualization

지도를 이용한 데이터 분석 과제
